##Restaurant Ratings Analysis - Power BI Dashboard
📌# Project Overview

This project is a Power BI Dashboard created to analyze restaurant ratings, customer preferences, and cuisine trends using multiple datasets. The dashboard provides interactive visualizations and insights that help understand customer behavior, restaurant performance, and popular cuisines.

The project uses CSV datasets containing restaurant details, ratings, cuisines, and consumer preferences.

📂 # Dataset Files

The project includes the following datasets:

restaurants.csv → Restaurant information
ratings.csv → Customer ratings data
restaurant_cuisines.csv → Cuisine categories for restaurants
consumers.csv → Consumer demographic details
consumer_preferences.csv → Customer preferences data
data_dictionary.csv → Dataset column descriptions
